
How to use the application:

1. Open Levinux
2. Navigate to the folder called "src" (cd src)
3. Run the web-app called "assessment.py (vim assessment.py)
4. Open your web browser
5. Type: "localhost:5000" in the web address bar

Extra:
1. In case you click on the link below the admin area, 
you can go back to the home page, by pressing the back button
on the browser or manually change the URL by typing "localhost:5000"
2. In case you accidently change the URL address, you need to type
"localhost:5000" or alternatively press back button on your browser to go
back to the home page.
